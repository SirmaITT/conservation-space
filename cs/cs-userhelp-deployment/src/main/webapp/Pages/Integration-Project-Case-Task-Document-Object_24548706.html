Sirma Platform : Integration Project-Case-Task-Document-Object

In this section are described the requirements for integration between business objects in the system - Project, Case, Task, Document.

Only new requirements are covered in the presented topics. For example:

  1. In the topic "Integrate Case in Project" are provided only the requirements in respect of the creation of a new case in a project and not about what is a case, in which status is it created, who could create a case, etc... which are already described (in "Create Case").
  2. In the topic "Integrate Task in Project" are provided the requirements for creating new single tasks in Project, Case or Task and not about the attributes of a task, the task status, etc. which are already described (in "Create Task").
