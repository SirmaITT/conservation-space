Sirma Platform : Start Project

Only after a project is officially started users are able to perform tasks on it.   

Only users with the role of a Project Manager could Start a Project. 

  • The user opens the project dashboard, reviews the comments on the project (1-2) and starts it  selecting the action "Start Project" (3-4) from the list.  
    • The action may be started from Personal Dashboard/ My Projects dashlet/ Action "Start Project"
    • The action may be started from Project details screen/ Actions /"Start Project'


  • The system displays the message "The project will be started and will become available for all assigned users. Yes/ NO" The user clicks "Yes".


  • The system opens the project details page, the user should fill in all mandatory data (1) and then click the button "Start Project" (2).

  • The system performs the following automatic actions: 
    • The project status changes to "In process" (1).   
    • Once started, the project could not be started again, and the button "Start Project" becomes invisible and new actions "Suspend Project" and "Complete Project" appear in the "Actions" list (2).
    • A notification that the projects is started is sent to each project assignee.
