Sirma Platform : Preview a Historic Version

Users have the ability to preview a historic (previous) version of the document without having to download the document content.

Preconditions

The document is opened for preview and has at least one historic version.

  • The user opens the document (1) and selects the action "Preview version" for an older version of the document (2-3).


  • The system opens the selected older version (1-2) of the document and all valid metadata details of the document for that version are visible in section "Properties" (3).
    • Â Available action on a historic version of a document is only "Download Document" (4).
    • On the screen the message "This is a historical document version, to get current click here" is displayed (5). When the link "here" in the message is selected, the system opens the current (last) version of the document.
