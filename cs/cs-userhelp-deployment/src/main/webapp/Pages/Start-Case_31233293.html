Sirma Platform : Start Case

Only after a case is officially started users are able to perform tasks on it.

Preconditions

  1. The user has relevant permissions to start the case.
  2. The case is in state "Approved".
  • The user starts the operation from:
    • Personal dashboard/ My Cases dashlet/ Action ""Start Case" for a chosen case
    • Case Dashboard/ button Actions / "Start Case"
    • Case Details page/ button Actions / "Start Case" (1-2)

  • The system displays the confirmation message "Are you sure you want to 'Start Case'?  Yes/ NO". The user clicks "Yes".

  • The system starts the case, performing the following automatic actions:
    • The case status changes to "In progress" (1).
    • Once started, the case could not be started again, and the action "Start Case" becomes invisible.
    • New actions are activated and appear in the "Actions" list (2): "Suspend Case" (3) and "Complete Case" (4).
    • Action "Delete Case" is deactivated and is not visible in the "Actions" list.
