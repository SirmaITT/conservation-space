Sirma Platform : Change Password

When the user is logged in the system the action "Change Password" is available in the drop down list next to the name of the user.

The password is between 5 and 30 characters long.

  • Change Password
  • Error Messages

Change Password

  • The user clicks "Change Password" (1-2) .

  • The system opens the screen with the following fields:
    • Current Password(1) - the field is mandatory and the user should enter his/ her current password
    • New Password (2) - the field is mandatory and the user should enter the new password
    • Confirm new Password (3) - the field is mandatory and the user should enter the same password as the one entered in ""New Password field
    • Action Change (4)  - confirms the new password
    • Action Cancel(5) - cancels the "Change password" action and transfers the user to his/ her Personal Dashboard.
  • The user enters the necessary data (1-2-3) and clicks "Change" (4).

  • If correct data is entered then the user's current password is changed with the one entered in "New Password" field.
    • A message is displayed - "Your password has been successfully changed" (1) and button OK (2) to close the screen.
    • The event for password change is registered in the audit log

Error Messages

When incorrect data is entered -  the system displays error messages which depend on the type of error. In such cases the password is not changed.
