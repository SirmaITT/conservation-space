Sirma Platform : View Project Profile

The Project Profile is a special view of the project where the user could view its details, view comments specific for the project and perform different actions on the project - edit, approve, start, stop, etc.  

Unlike to the project dashboard, which is a special view of all primary objects in a project and activities on them (task, sub-projects, resources, events, comments), the project profile is a view specific only for the project as an object.   

Any user who is assigned to the project, or any other system user who can browse the project (if the project is set to "Visible for all users") can view the Project Profile. 

  • The user starts the action from Project dashboard/ Project Profile dashlet/ View Project Profile (1-2-3).


  • The system opens the Project details page with the following sections:  

 Project Profile (1) - detailed information about generic project data and the specific for the project type sections. 

Comments (2) - in the section are listed all comments and replies on the project.  

 Project Members (3) - in the section are listed all resources assigned to the project with their roles in the project.  

 Actions (4) - in this section are listed all actions the current user could perform on the project at this specific state. For users with a role different from "'Project Manager" the section "Actions" is empty and hidden.
