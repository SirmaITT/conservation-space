Sirma Platform : Unlock Document

Preconditions

There is a locked document in the system which was locked by the currently logged user.

Only the user who has locked the document or an user with administrative permissions may unlock that document.

  • The user starts the action from:
    • Personal Dashboard/ My Documents dashlet/ Actions/ Unlock Document for a selected document
    • Case Dashboard/ Documents dashlet/ Actions/ Unlock Document for a selected document
    • Case Dashboard/ tab Documents/ Actions/ Unlock Document for a selected document (1-2-3)
    • Document preview/ Actions/ Unlock Document

  • The system unlocks the document (1).
    • the document can be edited by the users of the system, there is a possibility to edit the document properties or to update the version of the document.
    • the button  "Unlock Document"  becomes inactive (hidden) and the button "Lock Document" becomes active (visible)(2).
