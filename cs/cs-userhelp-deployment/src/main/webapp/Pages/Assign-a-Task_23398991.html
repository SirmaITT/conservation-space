Sirma Platform : Assign a Task

Here is described the process of the creation of a new task and its assignment to a selected user.

The creation and assignment of a new task is performed automatically :

    1. When starting a workflow - The user who starts the workflow selects an assignee whom the first task according the Workflow Diagram is assigned. (see topic: "Start Workflow')
    2. As a result from the completion of a task and the selection of a assignee of the next task. 
  • The user opens an active task, reviews the results from the previously completed task and decides to continue the workflow and:
    • enters Comment (1)
    • selects the assignee of the next task (2-3). The task assignee is chosen from a list of potential users or the task is assigned automatically based to certain rules.
    • select the action "Assign New Task" (4)

  • The task assigned to the user (here - Admin Admin) is completed (1-2).
  •  A new task is of the corresponding type according the Workflow Diagram is generated and send to the selected user (3).
  • The assignee of the task is given the rights of a "Collaborator" on the case.
