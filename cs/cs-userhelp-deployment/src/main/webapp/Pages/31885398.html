Sirma Platform : Create Task in Project, in Case or as Sub-Task

The user is able to create new single tasks in a project, in a case or as a sub-task in another task.

Preconditions 

  1. Only the task assignee and users with the role of "Manager" on the project could create sub-tasks of a task. 
  2. Sub-task could not be created in a task in status "Completed".
