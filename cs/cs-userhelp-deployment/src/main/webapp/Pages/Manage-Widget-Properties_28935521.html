Sirma Platform : Manage Widget Properties

The user may perform the action "Manage Properties" for each widget in the system.

The "Manage properties" action on a widget is active only when the iDoc is opened in Edit mode.

  • The user opens an iDoc in edit mode


  • The user selects the action Manage Properties from the drop down menu of actions of a selected widget (1-2).

  • A pop up screen opens with the widget specific details (1). For each widget there are different settings and properties which may be performed.
    • In the example the user selects "Object details to be displayed (2).

  • The user edits the properties, selecting more details for the object (case) to be displayed in the widget, selecting the correspondent check boxes (1-2-3-4-5)

  • The system opens the iDoc in edit mode. the changes are visible in the widget (1-2-3). The user saves the changes (4).

  • The added details (columns in the table) hold additional information (if available) for the object. The version of the iDoc has increased with 0.1.
