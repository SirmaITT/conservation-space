Sirma Platform : Move to Another Section

Only users with the respective permissions could move objects from one section to another.

The action is context dependent, which means that it could be executed only if an object is manipulated in a case.

  • To move an object between sections the user selects "Move to other section" action (1-2).

  • The system opens a pop up screen, where are listed all existing sections in the current case. It is possible that a case could have only one section.
    The user selects the section where the object link should be moved into (1-2).

  • The object is moved into the selected section of the case (1-2).
