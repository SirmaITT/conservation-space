Sirma Platform : View Document

The action "Preview Document" is performed for added documents and for iDocs.

When a link to a document in a case is selected, the system opens the document in preview mode with the following sections:

  1. Document with possibility to review its contents (1).
  2. Actions (2-3) that may be performed by the current user on the document.
  3. Annotations (4) (Comments) - comments and replies on the document.
  4. Properties (5) (Details) of the document - all metadata of the document (for which there is data in the system) defined as visible to the user.
  5. Relations (6) - a list with all relations of the documents and cases with the current document. The Name of the case is a link to the corresponding Case Dashboard.
  6. Share (7) - the user can copy the link after clicking on it with the right mouse button and then paste it in a MS document.
  7. Version History (8) of the document - a list with all older versions of the current document.

The content of the document is opened for preview paginated, in format A4.
