Sirma Platform : Add New Comment

Every user who has access to an object could comment on it. (Even the ones with view only access.)

  • An user could comment on every primary business object: task, workflow, sub-project/case, project, document, image, event, user, other domain specific objects
  • The user could comment on an object from:
    • Object landing page/ section Comments/ Add New Comment
    • "New Topic" /(New Comment) from every view of an object where there is a context menu with operations on the object (1-2).

  • The system opens an empty field (1) for entering a comment. The user enters the necessary data (2).
    • If the user clicks "Enter" from the keyboard the entered comment will be posted.
  • The user clicks the expand icon (3).

  • A pop-up screen opens with an option to enter the following details:
    • Title (1) - a short description of the topic of the comment. The field is mandatory.
    • Comment Body (2) - in this field the user adds the body of the comment. The system provides a text editor which could be used by the user for formatting the text. The field is optional.
    • Comment Type (3) - the category (purpose) of the comment , selectable from a drop-down menu (e.g., 'note to self', 'editorial question/suggestion, 'request assistance').

The user enters the necessary data (1-2-3-4).

  • The user clicks the button "Post".

  • The new comment is created and is associated to the respective object (in the example - case). The comment is created with the following additional metadata:
    • Created by: the user who has submitted the comment
    • Created on: time stamp (date and time) of when the comment is created 
    • The new comment is created in status"Open". Comment Status may be:open, resolved, closed.

 

  • The comment is visible in the "Comments" dashlet (1-2-3),  in the object landing page, and in the "Comments" section.

  • When a new comment is created it is in status "In Progress". 
  • A comment could have only one level of replies. See page "Reply on Comment".
