Sirma Platform : Download Document

The user performs the action "Download Document" in order to download the content of the document on his/ her local PC.

  • The user starts the action from:
    • Personal Dashboard/ My Documents dashlet/ Actions/ Download Document for a selected document
    • Case Dashboard/ Documents dashlet/ Actions/ Download Document for a selected document
    • Case Dashboard/ tab Documents/ Actions/ Download Document for a selected document (1-2-3-4).
    • Document preview/ Actions/ Download Document

When an attached document is downloaded:

  • it is downloaded on the user's local PC in a folder which is selected in the browser for downloading documents
  • it is downloaded in the same format as the format in which the document was attached in the case.

When an iDoc is downloaded:

  • it is downloaded on the user's local PC in a folder which is selected in the browser for downloading documents
  • it is downloaded in PDF format.
