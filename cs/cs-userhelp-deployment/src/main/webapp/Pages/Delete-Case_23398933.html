Sirma Platform : Delete Case

Preconditions

A case may be deleted only if it is in an active state.
A case may be deleted only by the user who created it or by an user with corresponding administrative permissions.

  • The user starts the operation from:
    • Personal Dashboard/ My Cases dashlet/ Actions/ Delete Case for a chosen case
    • Case Dashboard/  Actions/ Delete Case (1-2)
    • Case Dashboard/ Case details dashlet/ Actions/ Delete Case 
    • page Case details/ Actions/ Delete Case

  • The system opens a screen with confirmation massage "Are you sure you want to 'Delete Case'?". The user clicks the button "Yes".

  • The case is deleted and cannot be found in the system.
    • The active tasks/ workflows in the deleted case are cancelled.
    • All added documents and objects in the case are deleted also.
