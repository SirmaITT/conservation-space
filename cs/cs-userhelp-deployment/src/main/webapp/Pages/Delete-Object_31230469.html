Sirma Platform : Delete Object

An object could be deleted only by an user with the role of Administrator.

  • An object could be deleted by selecting "Delete Object" action from:
    • "Objects" dashlet/ action "Delete Object" for a selected object (1-2)

    • Object landing page/ "Delete object" (1-2)

 

  • The system displays the message "Are you sure that you want to continue with "Delete Object"? Yes/ No". The user selects "Yes".
