Sirma Platform : Text Editor

TheÂ  Text editor is used for formatting the text during entering comments in first class objects.

The text editor widget is implemented in each section by default and there is no need to be select from the widget palette.

Every time the user clicks in a section, the text editor menu becomes active and the user is able to: enter text, format the text (font, color, bullets), insert headings and tables, insert links to external links (links to external web pages) and insert links to external images and files.

The used text editor is TinyMCE (for demo see http://www.tinymce.com/tryit/inline.php ).


The default text editor functionality is extended with the following functions:

  • Insert link to first class system object - when the user selects this option (1) he/she is able to search (2) for any object in the system (project, case, document, image, etc.) and insert a link to it into the document.

  • Insert internal image - the user is able to search for images in the system and insert the whole image or its thumbnail into the document.

  • Insert object from "My favorites" - the user will be able to insert a link to an object without searching for it but directly inserting it from the "My favorites" (Not Ready)
