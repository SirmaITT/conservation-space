Sirma Platform : Checkbox

The insert check box option is enabled only when the iDoc is in edit mode.

  • The user selects the location where the widget should be positioned (1) and selects the Checklist widget from the Widgets palette (2).

The same action may be performed also by using drag and drop the widget in the content of the iDoc.

  • The widget is inserted in the selected section in the iDoc. The user enters title (1) and selects from the drop down menu (2) the action "Manage Properties" (3) for this widget.

  • The system opens the screen for entering details.
    • The check box will be inserted unselected in the selected section(1). There is a text field next to it where the user could enter desired text (2). There is an option to delete an entry (3).
    • To add a new check box in the checklist the user selects "Add new" (4) and a new check box will be added in the group.

    • When the user finishes with entering data he/ she selects the OK button

  • The data is entered in the section in the iDoc (1). The user clicks the button Save (2).

  • The system saves the changes in the document.
