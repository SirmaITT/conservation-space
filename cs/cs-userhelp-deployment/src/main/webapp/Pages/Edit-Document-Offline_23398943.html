Sirma Platform : Edit Document Offline

Preconditions

If a document is locked for edit by an user, then the action "Edit Document offline" is not active for all users until the document is unlocked by the user who has locked it.

  • The user starts the action from:
    • Personal Dashboard/ My Documents dashlet/ Actions/ Edit Document Offline for a selected document
    • Case Dashboard/ Documents dashlet/ Actions/ Edit Document Offline for a selected document
    • Case Dashboard/ tab Documents/ Actions/ Edit Document Offline for a selected documentÂ 
    • Document preview/ Actions/ Edit Document Offline (1-2)

  • When the action "Edit Document Offline" is selected for an attached document the following operations are carried out:
    • The message "This document is locked by <name of user>" (1) is displayed to the current user and to all other users who have opened the same document in preview.
    • The document is locked for edit and other users cannot edit it until the document is unlocked and can only view and download it.
    • The action "Download Document" is performed and the user has ability to edit the document's content and later on to upload it as a new version (2).
    • A limited set of actions are available for the user who started the action (3).
