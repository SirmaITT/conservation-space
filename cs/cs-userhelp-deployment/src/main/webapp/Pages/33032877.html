Sirma Platform : Upload Document/s in iDoc

New documents could be uploaded both in Edit and Preview mode of a document/ object.

The user could upload new documents from Actions/ Upload document in the landing page of an iDoc.

  • The user selects the action "Upload document" from the actions drop-down menu (1-2).

    • If the iDoc was not saved for the first time the system displays message and user is advised to save the iDoc first and then to upload documents in it.
    • the user should click "Cancel" and would be transferred to the landing page of the iDoc.

  • The pop-up screen for selecting documents opens (1) where the user:
    • Clicks the button "+Select File(s)" (2) .
    • Selects in the opened form (3) the necessary files for upload (4-5) from his/ her PC. The user click "Open" (6)
  • The selected files appear in the form for upload (7).

  • In the form for upload:
    • For each document there is a button Upload (1) and a button Remove(2). If selected the action applies only for the corresponding file.
    • There are buttons Remove All (3) and Upload All (6). If used the action applies for all selected files in the form.
    • For each document the user my select the Document Type from drop-down list (4), and enter a description of the document (5)
    • If the user selects the button "Cancel" (7), this would cancel the upload of the selected documents.
  • The user clicks "Upload All" (6).

  • The system displays message "Upload Complete"for the successfully uploaded items.
  • The user clicks "Cancel".

  • The selected documents are uploaded in the iDoc.
