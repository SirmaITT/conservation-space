Sirma Platform : Drag and Drop Section

The user is able to rearrange the sections of an iDoc using drag and drop action.

Sections could be moved only in a document in Edit mode. Sections could be moved only from the document tree view.

If automatic section numbering is on, then when rearranged by the user, the sections are automatically re-numbered again.

  • The user drags a selected section using the mouse.
    • He/she places the mouse cursor over the section title in the document tree and clicks and holds the left mouse button.

    •  By moving the mouse the user positions the section on the desired position inside the tree the user 'drops' the section in the desired position by releasing the mouse button.

  • When a section is moved, all links (links to other objects, external links, comments, etc.) to and from the section and the section content are moved too.
  • When the user drops a section between two other sections then it is positioned on the same level as the upper section (1-2-3).

  • When the user drops the section into another section then it becomes a subsection of the one in which is was dropped (1-2). After the user rearranges the sections in the document tree and is ready with the updates, he/ she selects the Save button and the system saves the changes.

  • If the user have turned on the automatic numbering on the iDoc, moving the section initiates the re-numbering to be performed automatically. In this case links and some properties of links may require automatic update.
