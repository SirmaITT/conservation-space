Sirma Platform : Object View

Each object in the system has a simple (compact) view and a detailed view.

The object in Detailed view is represented with large icon or thumbnail and with larger font (1).

The object in Compact (simple) view is represented with small icon and small font (2). The different object types are represented by different icons.

  • The documents and other objects, which have associations with images of type 'primary image (or thumbnail)' are represented with their thumbnails.
  • The activity objects (task, cases, workflows and projects) are represented with large icons.
  • (Not Ready) The users are represented with their photos both in compact and detailed view.
