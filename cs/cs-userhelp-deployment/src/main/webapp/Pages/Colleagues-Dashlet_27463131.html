Sirma Platform : Colleagues Dashlet

The use case describes the requirements for the content to be displayed on the Colleagues dashlet.  

  • The Colleagues dashlet is applicable for Project and Case dashboards, where the user could see all other team members which are working on the same project or case. 
    • On the Project dashboard in the dashlet are displayed all users which are assigned as resources to the project.
    • On the Case dashboard  in the dashlet are displayed only the users which have individual roles on the case or have tasks assigned on the case.
  • For each user in the Colleagues dashlet is displayed: User Name - First name and Family name (1) of the user, User role on the project (2), Job position


  • Next to each user in the dashlet there is an action "Comment". In the Comment screen in the "Attention to" field is automatically filled in the name of the user, who is selected. The comment is displayed then into the "Announcements and Messages" dashlet into the user personal dashboard.
