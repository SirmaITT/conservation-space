Sirma Platform : Project Details Dashlet

The use case describes the requirements for the content to be displayed on the Project details dashlet. 

This dashlet is applicable only to the Project dashboard and gives a brief information about what is the project about.

  • The information displayed in the dashlet is: Project ID, Project Title, Project Status, Project owner, etc.(1). Using the scroll (2) more information about the project may be reviewed. 
  • The user could perform the following actions on a project from the Project details dashlet:
    • View Project Profile (3) and using the link the Project profile landing page opens, where more actions on the project may be performed.
    • select action from Actions drop down list (4-5) that could be performed on the project.
