Sirma Platform : View Personal Dashboard


When a user logs in the system, automatically the first screen that is loaded is his/her Personal Dashboard.

Each user, registered in the system has a "Personal Dashboard".

The Personal Dashboard is the place where the user at any time could see at a glance a high-level overview of the status of work he/ she is performing (tasks, cases, documents) and the status of his/her projects .

The user could customize his/ her personal dashboard to suit his/ her individual needs. 

Any user (1) could access his/her personal dashboard with the direct link from the main menu "My dashboard" (2). 

The Personal Dashboard consist of the following dashlets: My Projects, Workflows, My Cases,  My Tasks, My Documents, Objects, My Media, My Comments, Recent Activities  

The user could customize his/ her personal dashboard and choose  which of the listed above dashlets to be visible or not.

 By default the following dashlets are visualized on the Personal Dashboard: My Cases, My Tasks, My Project, My Documents.
