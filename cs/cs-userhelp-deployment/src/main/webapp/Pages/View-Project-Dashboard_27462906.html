Sirma Platform : View Project Dashboard


Each project has a Project Dashboard.

The Project dashboard is the place where the Project Manager (PM) and each project member, could see in real time the status of a project, the recent activities on a project, overdue tasks, etc.

Only the PM  could customize the Project Dashboard to suit the project needs. 

When the user selects a project (selects the link of the project) always the project dashboard is opened. 

The Project Dashboard consist of the following dashlets: Project details,Workflows , Cases, Project Tasks, Documents, Objects, Media, Comments, Recent Activities, Colleagues, Discussion

 Which of the listed above dashlets to be visible or not could be customized by the user.

 By default the following dashlets are visualized  on the Project dashboard : Project Details, Colleagues, Project Tasks, Recent Activities.
