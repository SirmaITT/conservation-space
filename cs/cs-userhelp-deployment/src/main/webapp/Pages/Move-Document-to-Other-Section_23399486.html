Sirma Platform : Move Document to Other Section

The action is performed when the user wants to move a document from one section of a case in another section of the same case.

Preconditions

  1. The case in which the document is attached is in an active state.
  2. If the case has only one section, the operation is not permitted and there is no action "Move to other section" in the list with the possible actions for the document.
  • The user starts the action from:
    • Personal Dashboard/ My Documents dashlet/ Actions/ Move Document to other case for a selected document
    • Case Dashboard/ Documents dashlet/ Actions/ Move Document to other case for a selected document
    • page Case Details/ tab Documents/ Actions/ Move Document to other case for a selected document (1-2-3)
    • Document preview/ Actions/ Move Document to other case

  • The system opens a screen (1) with the list of all sections in this case. The user selects the target section (2) where the document should be moved and clicks the button "OK" (3).

  • The document (1) is moved in the selected section (2) of the same case .
    • The metadata details "Last modified on" and "Modified on" for the document are updated.
    • The document is not visible in the section from where the operation was started (3).
