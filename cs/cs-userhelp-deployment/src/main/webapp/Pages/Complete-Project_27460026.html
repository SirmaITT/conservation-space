Sirma Platform : Complete Project

Preconditions 

1. A project could be completed only when all tasks and sub-projects/ cases on a project are in end state - "Completed" or "Stopped". 
2. A project could be completed only by a user with the role of "Project Manager" 

  •   The user starts the action "Complete Project" from:
    • Project dashboard/ Project details dashlet/ Actions/ "Complete Project"
    • Project profile view/ Actions/ "Complete Project" (1-2)
    • Personal dashboard/ My projects dashlet/ "Complete Project" for a project


  • The system opens the project details screen in edit mode, the user fills in all mandatory data (1-2) and clicks the button "Complete Project" (2) .

  • The state of the project changes to "Completed" (1) and NO Actions (2) on the project are available.
