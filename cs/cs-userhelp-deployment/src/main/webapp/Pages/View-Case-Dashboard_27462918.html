Sirma Platform : View Case Dashboard

Each case has a Case Dashboard.

The Case Dashboard is the place where the user, could see in real time the status of a case, tasks on case, the recent activities on a project,comments and messages, etc.

Each user working on a case could customize the Case Dashboard to suit the his/ her personal needs. Actually the case dashboard is the working space of the user.

When the user selects a case (a link to that case), always the personalized case dashboard is opened. If no changes were made than the default case dashboard would be opened. 

The Case Dashboard consist of the following dashlets: Case Details, Tasks,  Documents, Workflows, Objects, Media, Comments, Recent Activities, Colleagues, Discussions, Linked Cases  

 (Not Ready) Which of the listed above dashlets to be visible or not could be customized by the user (see  Display/ Hide Dashlets).

 When a new case is created, it is created by default  with the following dashlets on the case dashboard: Case Details, Documents, Media, Workflows, Tasks, Recent Activities.
