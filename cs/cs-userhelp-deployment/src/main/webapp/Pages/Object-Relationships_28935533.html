Sirma Platform : Object Relationships

The Object Relationships widget in an iDoc contains information about all relationships that exist between the current object and other objects in the system.

  • The user selects the widget Relationships (1) from the widget palette and it is inserted in the selected section in the iDoc (2). The action may be performed by drag and drop.

  • The user enters the title of the widget (3). For each relationship is displayed:

    • Information for the object to which there is a relationship (4). The object is represented as a link.  When the link is selected -  the object landing page is opened.
    • Relation Type (5).

 

  • The widget displays also information for all objects that are referenced somehow in the document content.
    • If an object information is displayed in Object Data widget in the document - then automatically this object becomes visible in the list with object relationships (1).
    • If an object information is displayed in the iDoc Data table widget of the document - automatically all objects for which information is listed in the data table will be displayed in the object relationships widget.

 For example I create a report for my tasks in a project and in a data table I list information for each task, its status, Due date, etc. then all the tasks in the table will be listed in the relationships widget.

    • If a link to an object is inserted somewhere in the content of the document - then this object will be displayed also in the relationship widget.

For example I write a document and somewhere in the content I mention as a reference another document, then this document is listed in the relationships of my document with other objects.

    • If an image is inserted in the document content, then a relationship is created between the document and this image and it is displayed in the relationships widget (2).

  • For each object that is listed in the relationships widget in a document, there is opposite relationship showing the current document and the type of relationship.
