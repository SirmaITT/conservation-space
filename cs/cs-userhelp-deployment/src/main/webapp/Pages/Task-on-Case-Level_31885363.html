Sirma Platform : Task on Case Level

Preconditions 

  1. Only the task assignee and users with the role of "Manager" on the project could create sub-tasks of a task. 
  2. Sub-task could not be created in a task in status "Completed". 

Task on Case level

  • To create a  (associated with the work on a case) the user selects from:
    • Case Dashboard/ Tasks Dashlet/ Create Task button (1-2)
    • Case Dashboard/ Main Case Manu/ CreateTask button

  • The user selects the type of task, fills in the necessary data and clicks "Create task"
  • The newly created task is in relationship "Parent/ Child" with the case from which it is created. 


  •  The task is displayed in the project schedule as an entry of type "Task" (sub-task of the entry of type "Case").
