Sirma Platform : Task Notifications

 

  • When a user is assigned to a task, he is automatically notified via e-mail for the assignment.

  • The e-mail message for assignment of a user to a single task is generated automatically based on the following template:

    Subject: New task of type <Task type>

    Body:
    Hallo, <First Name>, a new task on project <Project ID,(Type), Title> has been assgined to you:
     Task ID: <Task ID>
     Task type: <Type>
     Title: <Title>
     Planned end date: <Planned end date>
     Creator: <First name, Family name of the user who created the task>
     Priority: <Priority>

    To view all task details click the link below:
    <link to the task>

    Reagrds,

    Conservation Space Administrator

 

  • When new pooled task is created, then all the users selected in the pool are automatically notified via e-mail.

  • The e-mail message for a pooled task is generated automatically based on the following template:

    Subject: New pooled task of type <Task type>

    Body:
    Hallo, <First Name>, there is new pooled task on project <Project ID,(Type), Title>, which you could claim to work on:
     Task ID: <Task ID>
     Task type: <Type>
     Title: <Title>
     Planned end date: <Planned end date>
     Creator: <First name, Family name of the user who created the task>
     Priority: <Priority>
     

    To view all task details click the link below:
    <link to the task>

    Reagrds,

    Conservation Space Administrator
