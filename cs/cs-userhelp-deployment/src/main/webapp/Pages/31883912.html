Sirma Platform : Suspend/ Restart Task

A task could be suspended and restarted more then once.

Preconditions 

  1. A task could be suspended only if in status "In Progress".
  2. A task could be suspended/ restarted only by the task assignee.
  3. A task could be restarted only if in status "On Hold"  (if it was suspended before that).

Suspend Task

  •   The user starts the operation for a task with status "Tn progress" (1) from:
    • Project dashboard/ My Tasks dashlet/ Actions/ "Suspend Task" 
    • Task Details/ Actions/ "Suspend Task" (2-3)
    • Case dashboard/ Tasks dashlet/ Actions/ "Suspend Task" 

  • The system opens the task details screen where:
    • The task status is changed to "On Hold" (1).
    • The action "Suspend Task" becomes inactive and invisible and the action "Restart Task" becomes active and visible (2-3).

Restart Task

  • The user selects the action "Restart Task" in the Task details screen. ( see screen above - 2-3).
    • The project status is changed back to "In Progress".
    • The button "Restart Project" becomes inactive and the button "Suspend Project" is activated.
