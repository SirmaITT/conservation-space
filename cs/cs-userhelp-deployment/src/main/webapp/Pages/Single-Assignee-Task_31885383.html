Sirma Platform : Single Assignee Task

The user is able to create new single tasks in a project, in a case or as a sub-task in another task. 

Basic task types are: single assignee task, pooled task, multi-instance task

Single Assignee Task

  1. The newly created task is assigned directly to the user, selected as the assignee of the task. Only one user could be selected.
  2. The task is displayed in the user's personal dashboard/ tasks dashlet.
  • An empty form for creation of a selected task type is opened. For this type of task only one task assignee may be selected. The user enters the necessary details of the task and clicks "Select".

  • The system opens the screen for selecting assignee (1). The user searches for an assignee entering part of he name and clicking "Enter" from the keyboard, and clicks the "+" button next to the name (2-3).

  • The selected assignee is moved into the right section (1). The user clicks "Save" (2).

  • The selected assignee is added in the "Assigned to:" field in the form for task creation (1). The user clicks "Create Task" (2).

  • The task is created and sent to the selected assignee. The task appears in the "My Tasks" dashlet of his/ her personal dashboard.
