Sirma Platform : Export iDoc

An iDoc may be exported in different formats like PDF, DOC, HTML and the user is be able to continue managing it.

Any user with access to the document can export it from the system.

  • The user opens the iDoc in preview mode and selects the action "Export".


  • (Not Ready) The system a screen opens where the user selects the type of export. The available types are: PDF or MS Word or HTML. By default is set PDF and the document will be exported with Portrait orientation and in A4 format.
    • If the user selects export in PDF then he/she is able to select additional setting on the exported document like:
      • page orientation - Landscape or Portrait
      • page format - A3, A4, Letter, etc
      • Header and Footer
    • After the type of export is selected and the additional settings are completed,  the user selects the export location of the document.
    • The document is exported in the selected format in the selected location with the applied settings.
