Sirma Platform : Attached To Relationship Between a Document or an Object with a Project or Case

When an object/ document is attached to the project or case, a relationship of type "Attached to" is automatically created between the document/object and the project/case.

  • Attach Object
  • Attach Document

Attach Object

  • The user may attach an object to a project.

 

  • The relationship Attached to can be seen in the Relationship tab of the object.

 

    • The inverse relation Has Attachment can be seen in the project's Relationships tab.

  • The user may attach an object to a case.

 

    • The relationship Attached to can be seen in the Relationships tab of the case.

 

  • The inverse relation Has Attachment can be seen in the case's Relationships tab.

Attach Document

  • In the same way (described above) the same relations are created when a document is attached to a project or to a case.
