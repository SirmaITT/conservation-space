Sirma Platform : Attach Document Directly in Project

The user is able to attach documents directly to the project.

Rules:

  1. In the library there is action “Attach document”
  2. Is Attached To/ Has Attachment relationship is created between the project and the document
  3. The user is able to set up the document types which may be attached to the project.
  • The user opens the project dashboard and selects the action “Attach Document”  from the list with all actions on project level (1-2-3).

  • The system opens the Document picker for selecting document/s (1). "Document" for object type is set in the search form (2).
  • The user selects the document/s to be attached from the whole system (3-4-5). More then one document may be selected.

  • The uploaded document appears in the Content dashlet (1) in the project dashboard.
  • The user opens the document (2).

  • In the Relations widget of the intelligent document (Relations section for documents) the automatically created relation "Is attached To" between the document and the project is displayed.

  • The user opens the tab "Relationships" (1) of the project.
  • The automatically created relation between the project and the document "Has Attachment" (2) is displayed.

  • The document is visible in the library of the project (1-2).
