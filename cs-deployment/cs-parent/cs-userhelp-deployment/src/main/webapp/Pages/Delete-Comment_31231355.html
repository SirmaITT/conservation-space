Sirma Platform : Delete Comment

  1. The creator of a comment may delete that comment.
  2. Replies may be deleted by the topic creator and by the reply creator.
  3. Comments associated with certain 1st class objects (e.g., projects, cases, documents) that have been locked or are in certain predefined states may not be deleted without special override permissions. (not Ready)
  4. The system admin and the project manager may delete any and all comments including replies.

Any Comment (topic) (including all associated metadata, body text, and replies) may be deleted after it has been initially saved. The topic may be deleted only if all of the replies to it are deleted also.

Any Reply to a Comment may be deleted after it has been initially saved.

  • The user opens the comment (1) and selects the "Delete" action from the drop-down menu (2-3).

  • The system opens a pop-up screen for confirmation. The user clicks "OK".

  • The selected comment is deleted.
