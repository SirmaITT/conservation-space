Sirma Platform : Search Codelists

The operation is performed from the Codelist Management Panel by users with administrator permissions.

  • The user opens the Codelist Management Panel from Main menu in SEIP/ Admin Tools.
  • The system opens the code lists management panel. From this panel the user is able to:
    • Upload a new code list (1)
    • Create a new code list (2)
    • Search for code list with entered search criteria (3)

The search form contains fields for searching for code lists where the user may enter search criteria:

      • Code list ID 
      • Description
      • Valid from - validity start date
      • Valid to - validity end date
      • Master Code list
      • Extra1

Additional search criteria may be selected after clicking the button "Show more search criteria" (4)

      • Extra2
      • Extra3
      • Extra4
      • Extra5
      • Comment

Rules:

  1. If some search criteria are defined by the user, the system performs search:
    • by part of the text - if a special character is entered (asterisk sign * -  this sign replaces zero or more characters)
    • by full text/exact match - if no special character is entered
  2. Search by criteria is case insensitive.
  3. If no criteria is entered and search is performed the system displays all code lists.
  • When the user enters the search criteria he/ she selects the button Search (5).
  • The system displays the results in a table view (6) ordered by default ascending, by code list identifier. The results in the table are paginated (7). The page size is defined in a configuration file.

Each time search is initiated and performed the first page of results is displayed.
