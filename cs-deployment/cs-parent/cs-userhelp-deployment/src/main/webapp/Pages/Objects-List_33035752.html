Sirma Platform : Objects List

The Objects List widget allows the user to search for objects and view the search result in the iDoc.

The user could select for the search result, when inserted into iDoc to be displayed in detailed view (this is the view of objects as displayed in the search result) or in compact view (this is the view of objects as displayed in the dashlets).

  • The user inserts in the iDoc the Objects Lists widget selecting it from the widgets section (1-2-3), enters name (4) and selects the action "Manage Properties"(5-6).

  • The form for selecting objects to be displayed in the widget (1).
    • The user chooses the view in which the objects to be displayed. By default the "Detailed view" option (2)  is selected.
    • The user selects option "Search for objects" (3)
      • The other option is "Manually select objects" . Using this option the user should define (check) manually  which objects from the search results to be displayed in the widget.
    • By default the Basic (4) search form is selected. The user may use also the Advanced search form to configure the query for the search.
    • The user enters search criteria (5-6) and clicks "Search" (7). The results from the search are displayed (8).
    • The user clicks the button "OK" (9).

  • The objects are displayed in the widget in detailed view.

  • In compact view the object are displayed as shown below.
    • The name and title of the objects are a direct link to open the corresponding object's landing page.
    • There is a tool-tip showing more information for the object

  • The user saves the document with the inserted widget. The objects according the selected view and criteria are displayed.


  •  If the system does not find objects from the search (according the entered search criteria), the message "No results found" appears in the widget and in the i-Doc.
