Sirma Platform : Access the Code List Management Panel

The access to the code lists management panel is from the main menu in SEIP/ Admin Tools.

  • Only user with administrator permissions has access to the code lists management panel from the main menu  Admin Tools/ Code List Server (1-2).

 For users without relevant permissions this option is hidden.

  • The system opens the code lists management panel. From this panel the user is able to:
    • Search for code list (1)
    • Upload a new code list from external file (2)
    • Create a new code list (3)
    • Search for code list with entered search criteria (4)
    • Manage codelists (5)
