Sirma Platform : Entity Recognition Based on Regular Expressions for Search Criteria

The user is able to define regular expressions for matching entity's properties in the definitions. In this way rules are defined based on which only some of the entity's property recognized in a documentÂ  and matching are automatically processed. Only over the matched documents actions are applied. These actions are defined in the rule definition.

Example:

The following documents are recognized in another document:

Doc1 with name: 123456.pdf
Doc2 with name: 123457.pdf
Doc3 with name: 12345Z.pdf

If the following regular expression is defined: 6d.pdf (six digits follwed by .pdf), then only the documents with names: 123456.pdf and 123457.pdf are related with the current document.
