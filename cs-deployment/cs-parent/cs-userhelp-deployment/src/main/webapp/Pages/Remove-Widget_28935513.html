Sirma Platform : Remove Widget

The user is able to remove widgets in iDocs.

The action is performed only when the iDoc is in Edit Mode.

All widgets in a document could be removed except the Title widget in the default document section.

  • The user selects the action Delete Widget from the drop down menu of actions in the widget (1-2).

Removing a widget from a section deletes only the widget component from the section. This operation does not delete any data. When a widget is removed from a section all created links from and to this widget are deleted.

  • The widget is removed from the content (1). The user selects the Save button (2).

  • The widget is removed  from the document.


NOTE: There are only a few widgets, like checklist, radio buttons, text fields, which when deleted from the section, the text associated with them is also deleted.
