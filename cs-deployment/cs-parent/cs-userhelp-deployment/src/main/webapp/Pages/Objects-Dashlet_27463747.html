Sirma Platform : Objects Dashlet

The Objects dashlet is context dependent and shows different content for the different dashboards.
In the content is displayed only information for the specific domain business objects.

  1. On the Personal dashboard are displayed objects created/ updated in the last 28 days by the user or such flagged as favorites (not ready).
  2. On the Project dashboard in the dashlet are displayed recently created/ updated/ associated objects (object created/ updated the last 28 days) in the project and all its sub projects/ cases.
  3. On the Subproject/ Cases dashboard in the dashlet are displayed recently created/ updated/ associated objects (object created/ updated the last 28 days) in the sub project/ case.
  • In the header of the dashlet is displayed the number of objects in the dashlet and the total number of all objects of the same type to which the user has access (1)The objects in the dashlet are sorted by default by date of create/ update and the latest ones are on the top. For each object in the Objects dashlet is diplayed: Icon, Object type, Object ID, Object Name, Created by, Created on (depending on the definition for the object). They all are a link (2) to the corresponding object landing page.
  • The tool tip (3) of each object displays more information about it depenging on the object's definition (Created by, Created on).

  • Next to each object in the dashlet there is a list with available actions (1-2) on the object (restricted by user role in the project and case), which in general include: Clone Object, Edit Object Details, Add Thumbnail etc.

  • The user is able to sort the objects in the dashlet (on Personal, Case and Project dashboard) by the following criteria:
    • Date created/ modified on - this is the default sort criteria, descending order (the items recently created/ updated are on the top)

    • Title

    • Object Type
    • Creator
  • The objects may be sorted in ascending or descending order.
  • In the Project dashboard the user is able directly from the Objects dashlet to create or attach objects in the project (1-2).

The newly created object  is created  in the object library and a relationship of type "Part of" is created with the project.

 

On Personal Dashboard/ My Objects Dashlet are displayed only objects recently created/ updated by the current user and objects associated with his/ her projects, cases, or tasks, that are in an “active” status (e.g., Submitted, Approved, In Progress, Suspended).

The following filters are applied on the dashlet (1-2):

  • Objects I am working on (default) - this is the default filter and when selected I see all objects that are associated with my projects,my cases, or my tasks, which are in an “active” status
  • Objects I've recently created - the filter displays only objects created by me in the last month
  • Objects I've recently updated - the filter displays only objects updated by me in the last month
  • My Favorites - shows all the items that the user has marked as favorites (only on My dashboard).

On Project Dashboard/ Objects Dashlet are displayed only objects associated with the current project and its cases.

The following filters are applied on the dashlet (1-2):

  • Objects I am working on (Default) - when this filter is selected I see all objects that are associated with my cases, that are in an “active” status (My cases are cases created by me, or cases with tasks assigned to me).
  • All objects - the filter displays all objects that are attached to the current project or are attached to any of the cases in the project
  • Objects I've created - the filter displays only objects created by me, that are associated with the project or its cases
  • Objects I've updated - the filter displays only objects updated by me, that are associated with the project or its cases

On Case Dashboard/ Objects Dashlet are displayed only objects associated with the current case.

The following filters are applied on the dashlet (1-2):

  • All objects (default) - this is the default filter and it displays all cultural objects that are attached to the current case.
  • Objects I've created - the filter displays only objects created by me, that are associated with the case
  • Objects I've updated - the filter displays only objects updated by me, that are associated with the case
