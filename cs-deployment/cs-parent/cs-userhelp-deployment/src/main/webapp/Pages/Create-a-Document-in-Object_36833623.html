Sirma Platform : Create a Document in Object

A user with the respective permissions can create a document in the context of a domain object.

An object is created in the system.

  • The user starts the action Create Document action from:
    • The domain object's landing page.

    • An Objects section within the case.

    • The Objects dashlet on the case/project/user dashboard

  • The system opens the Document creation screen where the user must select the document type from the Document type drop down menu in the Document creation window,may enter a description.
  • The user confirms the action.

  • The document form opens and the user must type the title of the document, may enter content and then save the document.

  • The new document is created as part of the domain object and in its Relationship widget a relation of type Part of with its parent object is created.

  • On the other hand, in the parent domain object a relation of type Has child is created with the new document.

 

  • The action is recorded in the Audit Log.
