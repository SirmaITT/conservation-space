Sirma Platform : Create New Object

Only users with respective permissions on a project/ case could create new objects. These could be: Manager, Collaborator or Contributor.

  • The user starts the action from:
    • Project dashboard/ 'Projects Objects' Dashlet/ Create New Object action
    • Main Menu/ Project Library/Objects Repository/ Create New Object action (NOT READY)
    • Case Dashboard/ Objects Tab/ Create New Object action (1-2-3)

  • The system opens the screen for selecting the object type. The user selects it from the drop down menu (1-2).

    • The user is transferred to the object landing page, where continues with entering the object type specific data(1-2-3). The user could Save or Cancel the object creation.
    • When ready the user clicks Save (4).

 

  • The system opens the newly created object in preview mode. If the object is created successfully then the following metadata are automatically generated (1-2):
    • Unique object ID 
    • Created by (1) - the name of the user who created the object
    • Created on (2) - Date: Time stamp of when the object is created into the system.
    • If the object is created directly into a case, then the case ID is saved in sections  "Relationships" (3) and "Related Cases" in the object landing page.
    • The object becomes visible in the Case/ Objects tab if created from case.

  • The action is recorded in the Audit Log.
