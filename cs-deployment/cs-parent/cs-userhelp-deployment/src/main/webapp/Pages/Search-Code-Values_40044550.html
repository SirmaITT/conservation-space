Sirma Platform : Search Code Values

  • The user with administrator permissions opens a code list in preview mode (1).
  • On the code list details form there is a section for searching code values in the code list. The user clicks the button "Search code values" (2) in the section.


  • The system opens a group of fields for searching for code values where the user may enter search criteria (1).
    • The form may be closed if the button "Hide code value search"(2) is selected.

Search may be filtered by the following criteria (fields):

  • Code
  • Description
  • Valid from - validity start date
  • Valid to - validity end date
  • Master Code value
  • Extra1

Additional search criteria may be selected after clicking the button "Show more search criteria" (3)

  • Extra2
  • Extra3
  • Extra4
  • Extra5
  • Comment

Rules:

  1. If some search criteria are defined by the user, the system performs search:
    • by part of the text - if a special character is entered (asterisk sign * -  this sign replaces zero or more characters)
    • by full text/exact match - if no special character is entered
  2. Search by criteria is case insensitive.
  3. If no criteria is entered and search is performed the system displays all code lists.
  • When the user enters the search criteria (4) he/ she selects the button Search (5).
    • If the user selects the button "Clear"(6) the form would be cleared and the user may enter new data for search.

Each time search is initiated and performed the first page of results is displayed.

 


  • The system displays the results from the search (1-2) in a table view (3) ordered by default ascending, by code. The results in the table are paginated (4). The page size is defined in a configuration file.
