View Project library

In every project there is a place called Library, where by default is displayed the content of the project (documents, tasks, cases, workflows, attached objects), including the content in all the cases in the project.

  • The user expands the Library (1) in the leftmost section of the project dashboard and selects "All" (2).

  • The Library opens. By default the Basic Search screen (1) is restricted to search by all objects in the context of the current project (2) (the name of the current project is displayed in the context field of the basic search).
    • The user may change the context in the default library search (2), but next time he opens the library, the same context (current project) would be displayed again.
    • By default in the search result are displayed all objects existing in the context of the current project (3) or in the cases part of the project, this includes also the attached domain specific objects or documents.
    • The configured facets (for Object Type, Created by, Created on, Modified by, Modified on) in the search screen are displayed together with results of the executed default query or other searches performed from the search screen in  Library (4).
    • The user has the ability to change basic with advance search (5).
    • The Saved searches functionality in the search screen is available (6).

Expand/ Collapse Search screen in the library

 

In order to increase the useful (working) area on the screen and to reduce the scrolling, when the Library of the project is opened the Search screen (basic search, advance search and saved searches) is collapsed by default and the user sees only the search result.

The user is able to expand the search screen and to view the search query at any time.

  • The user opens the project library (1) and sees the section Search Form (2) collapsed.
    • The content of the project's library is displayed in the search result (3).
  • The user chooses to open the Search Form (2).

  • The search form is opened.
    • The user has the ability to switch from basic search to advanced search.
