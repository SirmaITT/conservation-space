Sirma Platform : Browse Relationships

The action is performed when the user opens the Relationships screen where by default all existing relations for that object are displayed.

  •  The user could filter the existing relationships by relationship type, selecting it from a drop down list (1-2-3) or entering the type in the "Filter by type" field.

  • The system displays the results:

  • The user could filter the existing relationships by keyword of the object entering it in field "Filter by object".

  • The user enters text and the system immediately displays the result  lists only the rows of the table with references, in which the entered keyword exists (1-2-3).

If the user wants to see again all references, he/she deletes the text in the field.
