Sirma Platform : Complete Case

A case could be completed only by an user with the role of "Project Manager"  

  • The user starts the operation from:
    • Personal dashboard/ My Cases dashlet/ Action  "Complete Case" for a chosen case in active state.
    • Case Dashboard/ button Actions / "Complete Case"
    • Case Dashboard/ Case details dashlet/ button Actions / "Complete Case" (1-2-3)
    • Case Details page/ button Actions / "Complete Case" 

  • The system opens "Case Complete" screen and the user enters reason for completion (1) and clicks the button "Complete Case" (2).  


  • The system completes the case. The case status changes to "Completed" (1) 
    • automatically data is filled for "Completed on:" and "Completed by:" (2-3)
    • NO Actions on the case are available (2).
    • If there are tasks in the case that are not in a final state, then such tasks are cancelled automatically and their status is set to "Stopped" (5-6).
