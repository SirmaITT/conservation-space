Sirma Platform : Remove Section

Only the user created sections could be removed from the document.

The default section could not be removed.

Sections could be deleted only from a document in Edit mode.

  • The user opens a document in edit mode and  removes a section from the iDoc by one of these ways:
    1. Select the whole text in the section body together with the title and press button "Delete" from the keyboard.

      • The selected and deleted section is not visible in the Content Section. The user selects the Save button.

      • The document is saved without the selected and deleted section.

          2. Select the "Remove section" option from the section actions in the section tree pane  - for each section there is an option to remove section.

    • The user selects the button "X" (1) situated next to the name of the selected section and then clicks the Save (2) button.

    • The selected section is removed from the document.

          3. (NOT READY) Select the "Remove section" option from the section actions in the section body pane - for each section there is an option to remove the section.

  • When the section is removed from the iDoc, all links (relations) from and to the section are deleted. All comments on the section are deleted also.
