Sirma Platform : Saved and Predefined Searches

The user is able to save search configuration from Basic or Advanced search as a saved filter, specifying  title for it and later load and use it at any time.

All new filters have unique names - two filters with same name cannot exist.

Filter without Name cannot be saved - options "Update" and "Create new" become disabled.

Saved Searches

Save New Search

  • In the search landing page the user opens the "Saved Filters" functionality.


  • In the Basic or in the Advanced Search form (1) the user selects search criteria (2) and clicks the button "Search" (3).
  • The system displays the results in the list (4).

  • The user enters the name (1) and the description (2) of the filter. The field for the description is resizable.
  • The user selects the option to save the filter clicking the button "Create new" (3).

  • The search is saved (1) as a new filter and appears in the drop down list of saved searches (2). 
  • The list consists of two sections - Basic and Advanced.
  • The name of the filter (3) is in the section "Advanced" (4), because the search criteria were entered in the "Advanced "search form.

Note: When the user tries to create a filter with already existing name and error indicating that filter with the same name already exist is shown.

Load a Saved Search  

  • The Basic or the Advanced search form is opened.
  • The user loads a previously saved search by selecting it from the drop down menu of the field "Load existing filter" (1-2-3).

  • The selected saved search (1) with its name and description, here - basic (2) and its previously saved criteria (3) is loaded and displays the results from that search.

Reset
  • A saved search is loaded and the user clicks the button "Reset".

  • The system opens the empty search form of the the type of the previously loaded saved search (1).
    • The name and the description of the saved search are preserved. If the user clicks the button "X" (2), the fields for the name and description of the filter would be cleared.
    • The user may add search criteria and update the same saved search or create a new saved search (3/4)

Update a Saved Filter

  • The user loads a saved search (1). The search criteria are filled in the form (2). The system displays the results (3).

  • The user edits the criteria (changing/ adding/ deleting them) (1) and reviews the results(2-3).
  • The user changes the name and/ or the description of the search (4-5) and clicks the button "Update"(6) to save the changes of the filter.

  • The saved filter is updated with the new criteria, name and description (1-2-3-4).

Note: If the user does not change the name of the filter before clicking "Update" - the filter will be saved with the same name.

Saved Searches are found after search

  • The user is able to search for a saved filter as object of type "Saved Filter" in system (1).
  • The saved searches is listed in the search results (2).

Predefined Saved Searches

In the list of the existing filters (1) may be found saved searches marked with a star in front of the name (2). These are predefined searches and any user may use them.

  • The user selects a predefined search (1-2).

  • The search with its saved criteria is loaded (1-2) and the results are displayed (3).
  • The search criteria  and the comparison operators cannot be deleted. The user may only change the values of the predefined criteria.
  • The user has the option to add new criteria (4) and/ or save the search with a new name and / or description. The search would be saved as a new saved search ( not as a predefined search).
