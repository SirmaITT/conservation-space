Sirma Platform : Start Discussion on Case

Discussion may be started only if the case is in an active state.

Every user who has permissions to browse case may start new discussion for a case or make comments in an opened discussion.

Unlimited number of discussions may be started for each case.

  • Discussion for a case is started from: Case/ Case Dashboard/ Discussion dashlet. The user enters text in field (1) and clicks the button "Add" (2).

  • The discussion is started andÂ  appears in the Discussion dashlet in the case dashboard of any user with permissions to browse that case.
    Each comment displays the date when the comment was made and the user who made it (1).
    The comment is displayed below the date and the creator of the comment (2).

  • At any time users may add comments in the discussion (1-2). Using the time filters (3) the discussions and comments for other periods may be reviewed.
