Sirma Platform : Log Work on Task

  1. Each user (not only the task assignee), who has at least view permissions on a task, could log work on that task.
  2. Each Creator of a task, or Task Assignee is able to log time for other users, who've worked on the task.

In each task, there is a section Log Work (1), where the user could log the time spent working on a task.

  • The user opens the task and selects the button "Log work" (2).

  • The system opens the Log Work (1) screen and the user enters the following details:
    • Time SpentÂ (2) - the user should enter the time he/she spent in minutes, hours or days (30m, 2d, 4h, 4h 30 m, 3d 2h 25 m). The field is mandatory.
    • Start Date (3) - the user should enter the date when started working on the task. The date could be selected from the build-in calendar. The field is mandatory.
    • User Name - by default when Log Work option in a task is selected the name of the current user is displayed in the field. The User Name field is editable. Here the creator/assignee of the task may select any user from the drop-down list and log time for him/ her (4-5-6).

    • Work Description (1) - text field, in which the user enters a brief description of what he/she did. This is an optional field.
    • When ready, the user clicks the button Log (2).

        • If no user is selected in the corresponding field (1-2-3-4), the time is logged for the current user.

  • The system displays the task page.
    • The "Total time spent" field (1) in the task details is automatically calculated as sum of the logged times on the task for all users who worked on it and could not be changed.
    • The user name of the user who has logged time on the task and/ or the user name of the user for whom time was logged are automatically entered and could not be changed.
    • In the "Work Description " field for logged time for another user automatically is added text "Work logged by: < name>" (2) - here is the name of the user who logged time for another user .
    • There is an option for the user to edit the record (3) of time he/ she spent on the task.
    • There is an option for the user to delete his/ her record (4) of time spent on the task. The operation needs to be confirmed.

  • If the user selects the option to edit the record, he may change the Time Spent, Start Date or Work Description. There is no option to change the User Name.
