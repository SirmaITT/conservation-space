Sirma Platform : View Codelist Details

Code list details may be viewed when search results are listed.

  • The results from performed search are displayed (1). The user clicks on a row (2) from the search results to view the details for a selected codelist .

  • The system displays a new page with full details for the current version of the selected codelist (1). The form is in preview mode and the user cannot edit any of the fields.
    • The displayed data contains a list (table) with the code values in the codelist (2). The list is paginated (3).
    • More details for the codelist may be displayed when the button "Show more details" (4) is clicked.
    • The user is able to return to codelist browsing clicking the corresponding button (5).
