Sirma Platform : View Code Value

Code value details may be viewed when search results within a codelist are listed.

  • The user opens a codelist to view its code values (1-2).

  • The system opens the selected codelist (1) with its code values .
    • The code values within the codelist are displayed in a separate section (2) in a table view.The list of code values is paginated (3).
      For each of the code value rows the user is able to view the details of the code value if click somewhere on the row.
  • The user clicks on the row of a code value to view its details (4).

  • The system opens a popup screen with the full details for the current version of the selected code value. The form is in preview mode and I cannot edit any of the fields.

From this form the user may edit the code details (1) of the selected code value or close the form (2).
