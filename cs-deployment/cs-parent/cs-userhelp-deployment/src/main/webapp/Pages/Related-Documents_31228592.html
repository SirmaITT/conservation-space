Sirma Platform : Related Documents

The iDoc is opened in edit mode.

  • The user selects the "Related Documents" widget (1) from the widget palette and inserts it in the iDoc (2).
    • The Related documents widget is a sub-widget of Data table widget and displays all documents which are related with the current document/ object.
      • With predefined query which searches for all documents in relationship with the current idoc/ object .
      • With the following table columns: Enity (link to the document), Title, Description, Mime type, Created by, Created on (3).
  • The user adds a tittle (4)
  • There is action "Upload new document" (5) in the upper-right corner in the widget.

  • When the user selects the action "Upload new document", the Upload screen opens and the user is able to upload new documents from his / her PC.

    • selects a file/s from the PC
    • uploads the selected file/s
  • When the operation is completed clicks the "OK" button in the Upload screen.
  • The new file/s is added in the widget and in the corresponding documents section (here - Treatment reports) of the case (here - Test Case 1).
