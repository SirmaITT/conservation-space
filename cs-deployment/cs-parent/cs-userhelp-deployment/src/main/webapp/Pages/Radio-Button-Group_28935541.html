Sirma Platform : Radio Button Group

The user is able to insert a radio button group in an iDoc.

  • The user selects the "Radiobutton" (1) widget from the widget palette, drags it and drops it in a selected section in the iDoc (2).
  • The user inserts a title of the widget (3) and selects from the drop down menu (4) the action "Manage Properties" (5).

  • The system opens the Radio button (1) screen where the user insert the name of each button (2) in the group. Buttons are added by clicking the button "Add" (3).

Each entry may be deleted clicking the correspondent button "x" (4).


  • When the user is ready with the entries (1) he/she selects "OK" (2).

  • The system displays the document in edit mode.

The radio button group is enabled only when the document is in edit mode. The user selects a button from the group (1) and saves the changes in the iDoc (2).

 Only one of the radio buttons  in the group could be selected.

  • The system displays the document in preview mode.
