Sirma Platform : Media Dashlet

The dashlet is context dependent and shows different content on the different dashboards.  

  1. In the Personal dashboard in the My Media Dashlet are visible all the images created/ updated by the user and the images flagged as favorites.  
  2. In the Project dashboard in the Media dashlet   are visible all the images associated with the project.  
  3. In the Case dashboard, in the Media dashlet are visible all images associated with the case.

 In the Media dashlet are visible only images of type jpg, tiff, png, gif, bmp.

  • In the header of the dashlet is displayed the number of images in the dashlet and the total number of all images of the same type to which the user has access (1).
  • The images are visible in thumbnail view (2). For each image the image thumbnail and the title (3) are displayed. The title of an image is a direct link to the image landing page.
  • For each image the tool tip (4) displays more information about: Last Modified by, Modified on.


  • Next to each image there is list of actions that could be performed on it (1-2). The actions depend on the user permissions. In general they include: Upload New Version, Edit offline, Move, Delete etc 

  • The items in the dashlet  on all dashboards may be sorted by the following criteria, selectable from the drop down list (1-2):
    • Date modified on - this is the default sort criteria, descending order (the items recently created/ updated are on the top)

    • Title

    • Media type

    • Creator

  • For each combination of criteria the user is able to sort the items in the dashlet in ascending or descending order (3).

On Personal Dashboard/ My Media Dashlet are displayed only Images/ Audio/ Video associated with the projects оr cases of the current user, that are in an “active” status (e.g., Submitted, Approved, In Progress, Suspended).

The following filters are applied on the dashlet:

  • My active media files (default) - this is the default filter and it displays all Images/Audio/Video files uploaded/ attached or modified by the current user only in cases or projects in active state.
  • Images - the filter displays only images (png, jpeg, tiff, etc.) uploaded or modified by the current user.
  • Audio - the filter displays only audio files (mp3, aac, wma, etc) uploaded or modified by the current user.
  • Video - the filter displays only video files (mp4, avi, flv) uploaded or modified by the current user.
  • My Favorites - shows all the items that the user has marked as favorites (only on My dashboard).

On Project Dashboard/ Media Dashlet are displayed only Images/Audio/Video  files associated with the project оr any of its cases.

The following filters are applied on the dashlet:

  • My media (default) - the filter displays only the Images/ Audio/ Video files created/ uploaded/ attached or modified by the current user.
  • All media - the filter displays the media files created/ uploaded/ attached or modified by any project member
  • Images - the filter displays only images (png, jpeg, tiff, etc.) uploaded/ attached into the project/ cases
  • Audio - the filter displays only audio files (mp3, aac, wma, etc) uploaded/ attached into the project/ cases
  • Video - the filter displays only video files (mp4, avi, flv) uploaded/ attched into the project/ cases

On Case Dashboard/ Media Dashlet are displayed only Images/Audio/Video  files associated with the case.

The following filters are applied on the dashlet:

  • My media (default) - the filter displays only the Images/ Audio/ Video files created/ uploaded/ attached by the current user to the case.
  • All media - the filter displays the media files created/ uploaded/ attached or modified in the case by any project member
  • Images - the filter displays only images (png, jpeg, tiff, etc.) uploaded/ attached to the case
  • Audio - the filter displays only audio files (mp3, aac, wma, etc) uploaded/ attached to the case
  • Video - the filter displays only video files (mp4, avi, flv) uploaded/ attached to the case
