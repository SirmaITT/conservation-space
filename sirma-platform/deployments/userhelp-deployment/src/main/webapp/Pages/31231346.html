Sirma Platform : Filter/ Sort Comments

Comments may be filtered and sorted in the Object landing page/ Comments section or in the "Comments" dashlet.

  • Filter Comments
  • (Not Ready) Sort Comments

Filter Comments

  • Comment on an object could be filtered, when the user selects button "Filter" in the "Comments" section.

  • A sub-section opens where the user could choose to filter the comments from the following criteria:
    • Author - the user could choose only between the users who commented on the object
    • Keyword - the user could type a free text to search through the comments
    • From/ To Date - the user could enter "From date" and/ or "To date" when comment is created
    • Category - the user is able to filter comments with specific category
    • Status - the user is able to filter comments with specific status
    • Tag - the user is able to filter comments with specific tag
  • The user could filter comments by using one of the criteria or by using combination of two or more criteria. The user selects the criteria and clicks the button "Filter" (1-2).

  • The system displays the result/s (1).
  • The user clicks on the link to open the comment (2).
  • The system opens the comment with the selected criteria for review (3).
  • The user is able to enter a reply (4)

(Not Ready) Sort Comments

The system provides the following sort options on the Comments section for each object:

  • "Date Modified" - this is the default option. The latest created/ modified topics are on the top.
  • "User Name" - Sorts topics by the name of the creator of the topic.
  • "Title" - sorts topics by title. If the topic hasn't got a title then the text in the description is used for sorting.

Sorting of the results may be done in ascending or descending order based on each of the above listed criteria.
