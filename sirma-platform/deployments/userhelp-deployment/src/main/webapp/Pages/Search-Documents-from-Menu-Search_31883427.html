Sirma Platform : Search Documents from Menu Search

The Search operation allows the user to search among all documents or other system objects to which he/ she has access.

  • The user starts the search operation from his/ her Personal Dashboard/ Search (1-2)
  • The Basic Search screen opens and the user selects from the first drop-down menu the Object Type - Document (3-4).


  • The user selects other search criteria if needed (1-2-3). The following are available: Object type, Object relationship, Search context, Date Created From, Date Created To, Created By.
  • The user clicks the button "Search" (4)
  • The system displays the search results in a table view, paginated (5), which contain the following columns: Document Type,Â Document State, Version, Created by, date and time of creation. The Document Type, Document State and Version are a direct link to open the corresponding document.
  • The results may be sorted by different criteria (6) in ascending or descending order (7).
