Sirma Platform : Print Audit Log Search Results

The Audit Log Search Results may be printed by members of the administrators group.

  • The user searches in the audit log (1-2-3-4-5)
  • The system displays the results in the "Found activities" table (6).
  • The user prints the search results selecting the action "Print" (7) in the menu of the "Found activities" table.

  • The system opens the screen with the message "Do you want to print the hidden columns too?" for selecting options:
    1.  To print all columns from the search results - button "Yes" (1)
    2.  To print only those columns that have been selected to be displayed in the table - button "No" (2)

    • If the user selects "Yes", the system opens the print preview screen. The search results are shown in a table with all columns and data for the events (1). The printed file contains the entered search criteria and the number of the founded results.
      • The Print (2) pop-up screen appears where the user selects printer (3-4) and prints (5) the audit log search results.

    • If the user selects "NO", the system opens the print preview screen. The search results are shown in a table only with the columns and data for the events that have been selected to be displayed in the table (1). The printed file contains the entered search criteria and the number of the founded results.
      • The Print pop-up screen appears where the user selects printer and prints the audit log search results (2-3-4-5).

    • The printed file contains the entered search criteria.
