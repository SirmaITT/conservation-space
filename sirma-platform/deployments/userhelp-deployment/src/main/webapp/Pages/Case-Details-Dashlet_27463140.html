Sirma Platform : Case Details Dashlet

The dashlet is only applicable in the Case dashboard and gives brief information what is the case about. 

  • The information visible in the dashlet is: Case Id, Case Type, Case Title, Case Status,  Modified by, Modified on, etc.. (1).
  •  Using the scroll (2) more information about the case is displayed.
  • The user could perform actions on the case selecting from the Actions drop down menu (3-4).
