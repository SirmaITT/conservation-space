Sirma Platform : Documents Dashlet

The Documents (Content) dashlet is context dependent and the content is different in all three dashboards - Personal, Project and Case.

  • For each document in the dashlet the following information is displayed: Document Id, Document Title, Modified by, Modified on. They are a link which when selected,  the user is transferred to the document preview page. There is a tool tip which displays more information about the document (Last modified by, time stamp of last modification)

  • Next to each document there is list of actions that could be performed on it (1-2). The actions depend on the user permissions. In general they include: Upload New Version, Edit offline, Move, Delete etc 

  • The items in the dashlet may be sorted by the following criteria, selectable from the drop down list (1-2):
    • Date modified on - this is the default sort criteria, descending order (the items recently created/ updated are on the top)

    • Title

    • Document type

    • Creator

  • For each combination of criteria the user is able to sort the items in the dashlet in ascending or descending order (3).

On Personal Dashboard/ My Documents Dashlet are displayed only documents associated with the projects оr cases of the current user, that are in an “active” status (e.g., Submitted, Approved, In Progress, Suspended).

The following filters are applied on the dashlet:

  • All active documents (default) - this is the default filter and it displays all documents created/ uploaded/ attached or modified by the current user only in cases or projects in active state or in domain objects.
  • iDocs - this filter displays only iDocs created/ updated by the current user in projects or cases in active state
  • Images/Audio/Video - this filter displays only images/ audio/ video uploaded/ updated/ attached by the current user in projects or cases in active state or in domain objects
  • Documents - this filter displays only files (documents, not images/ audio/ video files) uploaded/ updated/ attached by the current user in projects or cases in active state or in domain objects

On Project Dashboard/ Documents Dashlet are displayed only documents associated with the current project or its sub-cases.

The following filters are applied on the dashlet:

  • My documents  -  this is the default filter and it displays only documents created/ uploaded/ attached or modified by the current user in the project or in any of its cases.
  • All documents -this filter displays the documents created/ uploaded/ attached or modified from any project member
  • iDocs - this filter displays only iDocs created/ updated in the project
  • Images/Audio/Video - this filter displays only images/ audio/ video uploaded/ attached in the project
  • Documents - this filter displays only files (documents, not media) uploaded/ attached in the project

On Case Dashboard/ Documents Dashlet are displayed only documents associated with the current case.

The following filters are applied on the dashlet:

  • My documents (default)- this filter displays only documents created/ uploaded/ attached or modified by the current user in the case.
  • All documents - this filter displays the documents created/ uploaded/ attached or modified in the case
  • iDocs - this filter displays only iDocs created/ updated in the case
  • Images/Audio/Video - this filter displays only images/ audio/ video uploaded/ updated in the case
  • Documents - this filter displays only files (documents, not media) uploaded/ updated in the case
